# Headings
Use `#` to define headings (similar to `<h1>` to `<h6>` in [[HTML]]).
```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

# Bold, Italics, and Strikethrough
```markdown
**Bold Text**  
*Italic Text*  
~~Strikethrough~~  
```

# Lists
- **Unordered List:** Use `-`, `+`, or `*`
- **Ordered List:** Use numbers
```markdown
- Item 1
- Item 2  
  - Sub-item  
  - Sub-item  

1. First item  
2. Second item  
   1. Sub-item  
   2. Sub-item  
```

# Links and Images
```markdown
[GitHub](https://github.com/)

![Alt Text](https://example.com/image.png)
```

# Blockquotes
Used for quoting text.
```markdown
> This is a blockquote.
>> Nested blockquote.
```
